Разработанный нами (Малика и Айна), бот, предоставляет возможность отслеживания настроения и продуктивности, с аналитикой каждой записи. Это удобный способ наглядно отслеживать свои ресурсы и испралять просечки, которые могут негативно сказаться на здоровье и бладотворности ваших занятий, тех или иных. 

Как запустить: 
1. Вы клонируете файлы из репозитория, переходите в VSCode и там начинаете запуск: 
Запускаете файл requimenets.tx - для наиболее бытсрой установки всех требующихся модулей
2. Все файлы sql вы подключаете к БД - чтобы они работали 
3. Запускаете файл bot.py 
4. Переходите по юзу: @@vhlbjfbot - это наш бот, токен которого уже указан в нашем коде
5. Вот и все! Можете пользоваться всеми функциями, предоставляемыми наим ботом!
